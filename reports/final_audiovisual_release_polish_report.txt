DUST ROOT CAUSE
The Stallion V3 already attached GameplayVisualEffectsPilot with a textured alpha billboard.
The Bavarian used the four shared legacy wheel emitters in scripts/vehicle_controller.gd.
Those emitters drew a bare QuadMesh without material, texture, transparency or fade ramp;
therefore each particle rendered as a perfectly opaque rectangular card. Large landing
fragments also used a comparatively large BoxMesh and could reinforce the block-like look.

DUST SOLUTION
- Replaced bare legacy quads with a generated 64x64 radial soft-alpha texture; no external asset.
- Added alpha ramp 0 -> soft peak -> low tail -> 0, random scale, initial angle and angular velocity.
- Reduced billboard dimensions, vertical component, velocity and lifetime; kept original activation
  condition (non-asphalt above speed threshold or meaningful slip).
- Preserved four per-wheel emitters and reduced front-wheel amount relative to rear wheels so the
  plume remains behind the car and does not cross the body/camera.
- Surface amount factors: ASPHALT 0.035, GRAVEL 0.48, SAND 0.68, DEEP_SAND 0.82.
- At 22 m/s measured rear amount ratios: 0.0292 / 0.4000 / 0.5667 / 0.6833.
- Stallion pilot: billboard reduced to 1.45x0.44, scale 0.15-0.48, lifetime 0.50 s,
  alpha/color per surface and randomized rotation.
- Gravel uses small soft puffs; sand uses warmer, lower soft trails; asphalt is nearly absent.
- Landing fragments reduced from 0.09x0.045x0.15 m to 0.045x0.022x0.075 m with scale 0.40-0.75.

ENGINE AUDIO ROOT CAUSE
AudioManager mapped instantaneous speed/throttle directly to pitch with no inertia and allowed
pitch up to 2.05. The loop was predominantly sinusoidal, with no gear-shift transient and no
vehicle profile, causing a rapid rise into a steady high-pitched tone.

ENGINE AUDIO SOLUTION
The shared AudioManager now consumes existing speed, throttle, simulated RPM, gear, airborne,
surface, boost and vehicle index telemetry. Physics and drivetrain values are unchanged.
- Frame-rate independent exponential RPM smoothing: rise response 2.15/s, release 4.8/s.
- Gear changes create a single 0.19 s pitch dip; recovery is progressive.
- One shared synthesis path creates a seamless multi-harmonic/subharmonic loop with 2 Hz low
  modulation, reducing pure-sine fatigue without external audio assets.
- Stallion profile: 64 Hz source, rough/heavy harmonic balance, pitch range 0.70-1.42.
- Bavarian profile: 76 Hz source, cleaner/compact balance, pitch range 0.76-1.50.
- Absolute runtime comfort clamp: 0.62-1.54.
- Load increases body/volume; airborne load multiplies engine level by 0.72.
- Boost adds only +0.035 engine level and +0.035 pitch plus the existing short secondary event;
  no persistent whistle remains after turbo_time reaches zero.
- Existing looped surface layer remains separate; short events are never retriggered every frame.
